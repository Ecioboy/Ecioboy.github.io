Install ArchLinux Course

详细请看 : Archwiki

Download ArchLinux

复制下面链接，找到你所对应的国家下载iso镜像文件
https://archlinux.org/download/

制作UEFI启动盘 (Make UEFI startup disk)

  • 这里只做推荐工具，制作请自行google/baidu
    • Windows : Rufus
    • Linux : dd

启动制作好的启动盘

开始安装

1.禁用reflector

systemctl stop reflector.service

2.确保是否为UEFI模式启动（若输出了一堆内容，则已在UEFI模式)

ls /sys/efi/efibootmgr

3.连接网络
  • 无线连接
    • rfkill list # 检查无线设备状态
    • rfkill unblock wifi #启用无线连接
    • ```
      iwctl #进入交互式命令行
      device list #列出设备名，比如无线网卡看到叫wlan0
      station wlan0 scan #扫描网络
      station wlan0 get-networks #列出网络 比如想CMCC-5AQ7这个无线
      station wlan0 connect CMCC-5AQ7 #进行连接 输入密码即可
      exit #成功后exit退出
    • ping www.baidu.com #测试网络连接
  • 有线连接
    • 一般插上网线就可以联网了.
4.更新系统时钟

timedatectl set-ntp true

timedatectl status

5.更换国内镜像源加快下载速度(把中科大 ustc 的或者清华的放在最上面。)

vim /etc/pacman.d/mirrorlist
